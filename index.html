Memo Git

Créer mon premier Repo

Créer un dossier: mkdir nom-du-dossier

Créer un fichier: touch nom-du-fichier

Ce placer dans le dossier avec la commande cd dans le term

  • Pour activer un dossier comme repository Git, il suffit de se placer dans ce dossier avec le term puis d'utiliser la commande git init.
  • Pour gérer un repository, Git génère un index de tous les fichiers dont il doit faire le suivi. Lorsque vous créez un fichier dans un repository, vous devez donc l'ajouter à l'index Git à l'aide de la commande git add nomDeVotreFichier.extension. Par exemple :
    git add nom-du-fichier.extension

Pour gagner du temps, on peut ajouter tous les fichiers dans le répertoire courant avec: git add .

Lorsque l'on doit modifier notre repository, nous devons demander à Git d'enregistrer nos modifications en faisant un git commit. L'option -m nous permet de lui envoyer un message décrivant les modifications effectuées, ce qui s'avèrera très utile par la suite. Par exemple :
git commit -m "Add index.html"

Les Branches

Créer une branche: git branch nom-de-la-branche

Basculer sur une branche: git checkout nom-de-la-branche

Supprimer une branche: git branch -d nom-de-la-branche
