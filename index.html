Mémo

Créer un dossier et/ou un fichier sur mon ordinateur

Créer un dossier avec la commande mkdir.

  • mkdir nomDuDossier

Créer un fichier avec la commande touch.

  • touch nomDuFichier

Ce placer dans le dossier avec la commande cd dans le terminal.

Pour activer un dossier comme repository Git, il suffit de se placer dans ce dossier avec le terminal puis d'utiliser la commande git init .

chemin/du/dossier/NomDuDossier$    git init

Vérifier l'état des fichiers avec la commande git status.

  • git status

Pour gérer un repository, Git génère un index de tous les fichiers dont il doit faire le suivi.
Lorsque vous créez un fichier dans un repository,
vous devez donc l'ajouter à l'index Git à l'aide de la commande git add nomDeVotreFichier.extension. Par exemple :

  • git add NomDuFichier

Pour gagner du temps, on peut ajouter tous les fichiers dans le répertoire courant avec:

  • git add .

Ajouter un commit avec la commande git commit -m .
Lorsque l'on doit modifier notre repository, nous devons demander à Git d'enregistrer nos modifications en faisant un git commit -m. L'option -m nous permet de lui envoyer un message décrivant les modifications effectuées, ce qui s'avèrera très utile par la suite.

  • git commit -m "Votre message"

Si nous faisons que mettre à jour un fichier que nous avions déjà ajouté, nous pouvons condenser les deux étapes du git add et du git commit de la façon suivante:

  • git commit -a -m "Votre message"

Pousser son travail sur le repository avec la commande git push origin master.

  • git push origin master

Vérifier les commits

Voir les commits qui ont été fait avec la commande git log.

  • git log

Modifier le message de mon dernier commitavec la commande git commit --amend -m .

... mais attention, on ne peut modifier ce message que si nous n'avons pas encore Pushé notre commit sur l'origine !
  • git commit --amend -m "VotreNouveauMessage"

Les branches

Créer une branche avec la commande git branch.

  • git branch nomDeLaBranche

Basculer entre les branches avec la commande git checkout.

  • git checkout nomDeLaBranche

Supprimer une branche avec la commande git branch -d .

  • git branch -d nomDeLaBranche
